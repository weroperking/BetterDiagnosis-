# prereq_01 — المصطلحات الطبية الأساسية

> **لماذا هذا الملف أول شيء؟**
> لأن المصطلح الطبي ليس كلمة عشوائية — هو جملة مضغوطة. من يفهم البنية يفهم المعنى تلقائياً حتى لو رأى الكلمة للمرة الأولى.

---

## 1. بنية المصطلح الطبي

كل مصطلح طبي تقريباً مكوّن من ثلاثة عناصر:

```
[ بادئة Prefix ] + [ جذر Root ] + [ لاحقة Suffix ]
```

**مثال:**
```
gastro  +  enter  +  itis
 معدة       أمعاء    التهاب
= التهاب المعدة والأمعاء = Gastroenteritis
```

## 2. البادئات الأكثر تكراراً

| البادئة | المعنى | مثال |
|---------|--------|-------|
| `anti-` | ضد | antibiotic = ضد الحياة (البكتيريا) |
| `hyper-` | فوق الطبيعي / زيادة | hypertension = ضغط مرتفع |
| `hypo-` | تحت الطبيعي / نقص | hypoglycemia = انخفاض السكر |
| `brady-` | بطيء | bradycardia = بطء ضربات القلب |
| `tachy-` | سريع | tachycardia = سرعة ضربات القلب |
| `dys-` | صعوبة / اضطراب | dyspnea = صعوبة التنفس |
| `a- / an-` | غياب / بدون | anemia = فقر الدم |
| `brady-` | بطيء | bradycardia = بطء نبضات القلب |
| `poly-` | كثير / متعدد | polypharmacy = تعدد الأدوية |
| `mono-` | واحد | monotherapy = علاج أحادي |
| `sub-` | تحت | sublingual = تحت اللسان |
| `trans-` | عبر | transdermal = عبر الجلد |
| `intra-` | داخل | intravenous = داخل الوريد |
| `inter-` | بين | intermittent = متقطع |
| `peri-` | حول | pericarditis = التهاب غلاف القلب |
| `pre-` | قبل | preoperative = قبل الجراحة |
| `post-` | بعد | postoperative = بعد الجراحة |
| `neo-` | جديد | neonatal = حديثي الولادة |
| `micro-` | صغير | microbial = ميكروبي |
| `macro-` | كبير | macrocyte = خلية دم كبيرة |

## 4. اللواحق الأكثر أهمية لفني الصيدلية

### لواحق الأمراض
| اللاحقة | المعنى | مثال |
|---------|--------|-------|
| `-itis` | التهاب | hepatitis, gastritis |
| `-osis` | حالة / تليّف / تراكم | fibrosis, cirrhosis |
| `-emia` | في الدم | anemia, bacteremia |
| `-uria` | في البول | glycosuria, hematuria |
| `-algia` | ألم | neuralgia, myalgia |
| `-pathy` | مرض / اضطراب | neuropathy, nephropathy |
| `-megaly` | تضخم | hepatomegaly, splenomegaly |
| `-rrhea` | تدفق / إفراز | diarrhea, rhinorrhea |
| `-plegia` | شلل | hemiplegia, paraplegia |
| `-trophy` | نمو / تغذية | hypertrophy, atrophy |

### لواحق الأدوية — **مهمة جداً لفني الصيدلية**
| اللاحقة | فئة الدواء | أمثلة |
|---------|-----------|-------|
| `-cillin` | بنسلينات | amoxicillin, ampicillin |
| `-cycline` | تتراسيكلينات | doxycycline, tetracycline |
| `-mycin / -micin` | أمينوغليكوزيدات أو ماكروليدات | gentamicin, azithromycin |
| `-oxacin / -floxacin` | فلوروكينولونات | ciprofloxacin, levofloxacin |
| `-azole` | مضادات فطرية أو بروتون بمب | fluconazole, omeprazole |
| `-olol` | حاصرات بيتا | atenolol, metoprolol, propranolol |
| `-pril` | مثبطات ACE | enalapril, lisinopril, captopril |
| `-sartan` | حاصرات ARB | losartan, valsartan, irbesartan |
| `-vastatin` | ستاتينات | atorvastatin, simvastatin |
| `-dipine` | حاصرات قنوات الكالسيوم | amlodipine, nifedipine |
| `-pam / -lam` | بنزوديازيبينات | diazepam, alprazolam, clonazepam |
| `-tidine` | حاصرات H2 | ranitidine, famotidine |
| `-mab` | أجسام مضادة وحيدة النسيلة | adalimumab, rituximab |
| `-nib` | مثبطات كيناز | imatinib, erlotinib |
| `-gliptin` | مثبطات DPP-4 (سكري) | sitagliptin, saxagliptin |
| `-formin` | بيغوانيدات (سكري) | metformin |
| `-oxetine` | مثبطات استرداد السيروتونين | fluoxetine, paroxetine |
| `-triptan` | مضادات الصداع النصفي | sumatriptan, rizatriptan |

> 💡 **القاعدة الذهبية:** إذا رأيت اسماً دوائياً لا تعرفه، انظر للاحقته أولاً. ستعرف فئته فوراً.

## 5. المصطلحات السريرية اليومية في الصيدلية

### الجرعة والتكرار
| المصطلح | المعنى |
|---------|--------|
| Loading dose | جرعة تحميل أولية عالية لتحقيق تركيز سريع |
| Maintenance dose | الجرعة المستمرة للحفاظ على التركيز |
| PRN (pro re nata) | عند الحاجة — لا جدول ثابت |
| QD / OD | مرة واحدة يومياً |
| BID | مرتان يومياً |
| TID | ثلاث مرات يومياً |
| QID | أربع مرات يومياً |
| Q8H / Q12H | كل 8 أو 12 ساعة |
| HS (hora somni) | عند النوم |
| AC (ante cibum) | قبل الأكل |
| PC (post cibum) | بعد الأكل |

### مصطلحات الفارماكولوجيا الأساسية
| المصطلح | التعريف البسيط |
|---------|--------------|
| Half-life (t½) | الوقت اللازم لانخفاض تركيز الدواء في الدم إلى النصف |
| Bioavailability | النسبة التي تصل فعلاً إلى الدورة الدموية من الجرعة المأخوذة |
| Onset of action | الوقت حتى يبدأ الدواء في العمل |
| Peak effect | الوقت الذي يصل فيه الدواء لأعلى تأثير |
| Duration of action | كم يستمر تأثير الدواء |
| Contraindication | موانع الاستخدام — لا يُعطى في هذه الحالات |
| Adverse effect | أعراض جانبية غير مرغوب فيها |
| Drug interaction | تفاعل بين دواءين أو أكثر يغير تأثيرهما |
| Tolerance | تراجع الاستجابة للدواء مع الوقت |
| Dependence | اعتماد الجسم على الدواء |
