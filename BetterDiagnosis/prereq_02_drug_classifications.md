# prereq_02 — تصنيفات الأدوية

> **لماذا يهمك هذا؟**
> لأن كل يوم في الصيدلية ستتعامل مع مئات الأدوية. بدون تصنيف واضح في ذهنك، ستغرق. بوجوده، كل دواء جديد تراه يجد مكانه تلقائياً.

---

## 1. نظام ATC — التصنيف الدولي المعتمد

**ATC = Anatomical Therapeutic Chemical Classification**
نظام تصنيف منظمة الصحة العالمية (WHO). كل دواء معتمد في العالم له رمز ATC.

### الهيكل الهرمي — 5 مستويات

```
المستوى 1: الجهاز التشريحي (حرف واحد)
    └── المستوى 2: المجموعة العلاجية الرئيسية (رقمان)
            └── المستوى 3: المجموعة الفرعية العلاجية (حرف)
                    └── المستوى 4: المجموعة الكيميائية (حرف)
                            └── المستوى 5: المادة الفعالة (رقمان)
```

**مثال: الميتفورمين (علاج السكري)**
```
A       → الجهاز الهضمي والأيض
A10     → أدوية السكري
A10B    → أدوية السكري (خافضات السكر الفموية)
A10BA  → البيغوانيدات
A10BA02 → Metformin ✓
```

| الرمز | الجهاز / النظام | أمثلة سريعة |

| **A** | الجهاز الهضمي والأيض | ميتفورمين، أوميبرازول، إنزيمات الهضم |

## 3. التصنيف العملي في الصيدلية اليومية

### المجموعة N — الجهاز العصبي (الأكثر تنوعاً وأهمية)

| الفئة الفرعية | الاستخدام | أمثلة |
|--------------|---------|-------|
| N01 — تخدير | التخدير العام والموضعي | ليدوكايين، كيتامين |
| N02A — أوبيويدات | ألم شديد | مورفين، كودين، ترامادول |
| N02B — مسكنات لا أوبيويد | ألم خفيف-متوسط | باراسيتامول، أسبرين |
| N03 — مضادات الصرع | الصرع، بعض الألم العصبي | فينيتوين، كاربامازيبين |
| N04 — مضادات باركنسون | داء باركنسون | ليفودوبا، براميبيكسول |
| N05A — مضادات الذهان | الفصام، الهوس | هالوبيريدول، أولانزابين |
| N05B — مزيلات القلق | القلق، الأرق | ديازيبام، ألبرازولام |
| N05C — منومات | الأرق | زولبيديم، ترازودون |
| N06A — مضادات الاكتئاب | الاكتئاب، القلق المزمن | فلوكسيتين، سيرترالين |
| N06B — منبهات الجهاز العصبي | ADHD | ميثيلفينيدات، أتوموكسيتين |

### المجموعة C — القلب والأوعية الدموية

| الفئة | الاستخدام | أمثلة |
|-------|---------|-------|
| C02 — خافضات الضغط | ضغط الدم المرتفع (خط ثاني) | هيدرالازين، ميثيلدوبا |
| C03 — مدرات البول | الضغط، القصور القلبي، الوذمة | فيوروسيميد، هيدروكلوروثيازيد |
| C07 — حاصرات بيتا | الضغط، الذبحة، ضربات غير منتظمة | أتينولول، ميتوبرولول |
| C08 — حاصرات قنوات الكالسيوم | الضغط، الذبحة | أملوديبين، ديلتيازيم |
| C09A — ACE inhibitors | الضغط، القصور القلبي | ليزينوبريل، إنالابريل |
| C09C — ARBs | الضغط (بديل ACE بلا سعال) | لوسارتان، فالسارتان |
| C10 — خافضات الدهون | الكولسترول | أتورفاستاتين، رسيوفاستاتين |

## 4. التصنيف حسب آلية العمل (Mechanism of Action)

هذا التصنيف مهم لفهم **لماذا** يعمل الدواء، وليس فقط **ماذا** يعالج.

| الآلية | المعنى المبسط | مثال |
|--------|-------------|-------|
| Agonist | يحفز المستقبل ويفعّله | سالبوتامول يحفز مستقبلات بيتا-2 في الرئة |
| Antagonist | يرتبط بالمستقبل ويمنع تفعيله | أتينولول يمنع مستقبلات بيتا-1 في القلب |
| Enzyme inhibitor | يثبط إنزيماً معيناً | ACE inhibitors تثبط إنزيم ACE |
| Channel blocker | يسد قناة أيونية | أملوديبين يسد قنوات الكالسيوم |
| Reuptake inhibitor | يمنع إعادة امتصاص ناقل عصبي | فلوكسيتين يمنع إعادة امتصاص السيروتونين |
| DNA/RNA disruption | يتدخل في تكاثر الخلية | معظم المضادات الحيوية والكيماوي |

## 5. تصنيف الأدوية حسب درجة الخطورة في الحمل

نظام FDA (الأمريكي) — لا يزال مرجعاً شائعاً رغم تحديثه:

| الفئة | المعنى | مثال |
|-------|--------|-------|
| **A** | آمن تماماً — دراسات بشرية سليمة | حمض الفوليك |
| **B** | آمن نسبياً — آمن في الحيوانات | باراسيتامول، أموكسيسيلين |
| **C** | حذر — لا دراسات بشرية كافية | معظم الأدوية الشائعة |
| **D** | خطر مثبت — إلا عند الضرورة القصوى | فينيتوين، بعض الستيرويدات |
| **X** | ممنوع تماماً في الحمل | ميثوتريكسات، أيزوتريتينوين (روأكيوتان) |

## 6. تصنيف المخدرات والمواد الخاضعة للرقابة

في معظم الدول المواد مصنفة حسب درجة خطورتها وإمكانية الإدمان:

| الجدول | المعنى | أمثلة |
|--------|--------|-------|
| جدول I | ممنوع — لا استخدام طبي معتمد | الهيروين، LSD |
| جدول II | استخدام طبي + خطر إدمان عالٍ | مورفين، أوكسيكودون، ريتالين |
| جدول III | استخدام طبي + خطر متوسط | كودين بجرعات عالية، كيتامين |
| جدول IV | استخدام طبي + خطر منخفض | ديازيبام، ألبرازولام، زولبيديم |
| جدول V | خطر محدود جداً | سيروب السعال بالكودين (تركيزات منخفضة) |

> ⚠️ **تنبيه:** في الصيدلية، أي دواء جدول II أو III يتطلب وصفة خاصة وتسجيلاً في السجل الخاص بالمخدرات. لا تتجاوز هذه القاعدة أبداً.
