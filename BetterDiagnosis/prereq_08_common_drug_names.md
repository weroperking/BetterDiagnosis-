# prereq_08 — أسماء الأدوية الشائعة: الجنيسي والتجاري

> **القاعدة:** الاسم الجنيسي هو هويتك المهنية. الاسم التجاري هو ما يعرفه المريض. أنت بحاجة لكليهما.

---

## 1. الفرق بين الجنيسي والتجاري

| | الاسم الجنيسي (Generic) | الاسم التجاري (Brand) |
|-|------------------------|----------------------|
| **التعريف** | الاسم الدولي للمادة الفعالة | اسم الشركة المصنّعة للمنتج |
| **الكتابة** | بحرف صغير | بحرف كبير أو علامة ® |
| **المثال** | paracetamol | Panadol®, Tylenol®, Adol® |
| **السعر** | أرخص عموماً | أغلى |
| **التوفر** | قد يكون متعدد المصادر | شركة واحدة |
| **الفعالية** | متكافئة بيولوجياً | نفس المادة الفعالة |

## 4. الأدوية التي تُخلط أسماؤها — تحذير

هذه الأدوية متشابهة في الاسم لكن مختلفة تماماً:

| الزوج المتشابه | الفرق |
|--------------|------|
| **Losartan** vs **Lovastatin** | ARB للضغط vs ستاتين للكوليسترول |
| **Celebrex** vs **Celexa** | مسكن NSAIDs vs مضاد اكتئاب |
| **Hydroxyzine** vs **Hydroxychloroquine** | مزيل قلق vs روماتويد/ملاريا |
| **Zantac (Ranitidine)** vs **Xanax (Alprazolam)** | حموضة vs قلق/بنزوديازيبين |
| **Zyrtec** vs **Zyprexa** | حساسية (Cetirizine) vs ذهان (Olanzapine) |
| **Metformin** vs **Metronidazole** | سكري vs مضاد حيوي/طفيليات |
| **Vincristine** vs **Vinblastine** | كيماوي — جرعات مختلفة تماماً |

> 🚨 كل الأزواج أعلاه تسببت في حوادث طبية موثّقة. إذا شككت في الاسم — تأكد قبل الصرف.
