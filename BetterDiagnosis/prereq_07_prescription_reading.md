# prereq_07 — قراءة الوصفة الطبية

> الوصفة الطبية هي الوثيقة القانونية التي تعمل بها كل يوم. قراءتها بشكل صحيح ليست مهارة — هي **واجب مهني**.

---

## 1. مكوّنات الوصفة الطبية الكاملة

```
┌──────────────────────────────────────────────┐
│  اسم الطبيب + تخصصه + رقم ترخيصه            │
│  عنوان العيادة + رقم الهاتف                   │
├──────────────────────────────────────────────│
│  اسم المريض:              التاريخ:            │
│  العمر:                   الوزن (للأطفال):   │
├──────────────────────────────────────────────│
│  ℞  (Rx)                                      │
│                                               │
│  1. اسم الدواء + الجرعة + الشكل الصيدلاني    │
│     sig: التعليمات (متى، كيف، كم مرة)         │
│     Disp: الكمية المصروفة                      │
│     Refill: هل يُجدَّد وكم مرة؟              │
│                                               │
├──────────────────────────────────────────────│
│  توقيع الطبيب                                 │
└──────────────────────────────────────────────┘
```

### 📐 وحدات القياس

| الرمز | المعنى |
|------|--------|
| `mg` | ملليغرام |
| `mcg` أو `μg` | ميكروغرام (1/1000 ملليغرام) |
| `g` | غرام |
| `kg` | كيلوغرام |
| `mL` | مللتر |
| `L` | لتر |
| `IU` | وحدة دولية (International Unit) |
| `mEq` | ميلي مكافئ (للكهارل) |
| `mmol` | ميلي مول |

| `ppm` | جزء في المليون |

> ⚠️ **تحذير خطير:** `μg` و`mcg` تعني نفس الشيء. لكن إذا كُتبت `μg` وقُرئت خطأً كـ `mg` → الجرعة أكبر 1000 مرة! لهذا كثير من الأنظمة تطلب كتابة `mcg` بوضوح.

## 3. أخطاء القراءة الأكثر خطورة

هذه الأخطاء تسببت بحوادث حقيقية وفيّات في الصيدليات:

| الكتابة المشكوك فيها | الخطأ الممكن | القراءة الصحيحة |
|--------------------|------------|----------------|
| `U` (لـ Units) | قُرئت `0` → الجرعة × 10 | اكتب "units" بالكامل |
| `IU` (International Units) | قُرئت `IV` → وريدي | اكتب "international units" |
| `μg` | قُرئت `mg` → × 1000 | اكتب `mcg` |
| `.5 mg` (نقطة قبل الرقم) | قُرئت `5 mg` → × 10 | اكتب دائماً `0.5 mg` |
| `1.0 mg` (صفر بعد النقطة) | قُرئت `10 mg` → × 10 | اكتب `1 mg` بدون صفر |
| `qd` أو `qod` | يُخلط بينهما بسهولة | اكتب "daily" أو "every other day" |
| `MS` | يعني مورفين سلفات أو ماغنيسيوم سلفات | يجب التوضيح |
| `MTX` | ميثوتريكسات — جرعة خاطئة قاتلة | اكتب الاسم كاملاً |

## 4. كيف تقرأ الوصفة خطوة بخطوة

```
الخطوة 1 — تحقق من هوية المريض
  └─ الاسم + تاريخ الميلاد / العمر

الخطوة 2 — تحقق من صلاحية الوصفة
  └─ التاريخ + توقيع الطبيب + رقم الترخيص

الخطوة 3 — اقرأ اسم الدواء بعناية
  └─ هل تعرفه؟ هل الخط واضح؟ إذا شككت — اتصل بالطبيب

الخطوة 4 — تحقق من الجرعة
  └─ هل هي منطقية للعمر والوزن؟ هل تتجاوز الحد الأقصى؟

الخطوة 5 — افهم التعليمات (Sig)
  └─ ترجم الاختصارات، تأكد أن التعليمات واضحة للمريض

الخطوة 6 — راجع التفاعلات الدوائية
  └─ خاصة إذا كانت وصفة متعددة أو مريض مزمن

الخطوة 7 — تحقق من التاريخ الانتهاء وموانع الاستخدام
  └─ حمل، حساسية، أمراض مصاحبة

الخطوة 8 — اشرح للمريض
  └─ متى، كيف، ماذا يتوقع، متى يتصل بالطبيب
```

## 5. أمثلة وصفات حقيقية وترجمتها

### مثال 1 — بسيط
```
Amoxicillin 500 mg Cap
Sig: 1 cap TID × 7 days PC
Disp: 21 caps
NR
```
**الترجمة:** أموكسيسيلين 500 ملغ — كبسولة واحدة ثلاث مرات يومياً لمدة 7 أيام بعد الأكل — صرف 21 كبسولة — لا يُجدَّد.

---

### مثال 2 — متوسط
```
Metoprolol 50 mg Tab
Sig: ½ tab BID × 2 weeks, then 1 tab BID
Disp: 60 tabs
Refill × 2
```
**الترجمة:** ميتوبرولول 50 ملغ — نصف قرص مرتين يومياً لأسبوعين، ثم قرص كامل مرتين يومياً — 60 قرصاً — يُجدَّد مرتين.

> 💡 **ملاحظة:** الجرعة تبدأ منخفضة ثم ترتفع = Titration (معايرة تدريجية)، شائعة في حاصرات بيتا وأدوية الضغط.

---

### مثال 3 — للأطفال
```
Amoxicillin 250 mg/5 mL Susp
Sig: 7 mL TID × 10 days AC
Disp: 150 mL
NR
Shake well before use
```
**الترجمة:** معلّق أموكسيسيلين — 7 مللتر 3 مرات يومياً لمدة 10 أيام قبل الأكل — يُرجّ جيداً — لا يُجدَّد.

---

### مثال 4 — متعدد الأدوية (Polypharmacy)
```
1. Metformin 500 mg Tab    — 1 tab BID CC
2. Atorvastatin 20 mg Tab  — 1 tab QD HS
3. Amlodipine 5 mg Tab     — 1 tab QD AM
4. Aspirin 100 mg Tab      — 1 tab QD CC
```
**الترجمة:**
1. ميتفورمين — قرص مع الأكل مرتين يومياً
2. أتورفاستاتين — قرص مرة يومياً عند النوم (الكبد يصنع الكوليسترول ليلاً)
3. أملوديبين — قرص صباحاً
4. أسبرين — قرص مع الأكل يومياً

## 6. متى تتصل بالطبيب وترفض الصرف؟

| الموقف | الإجراء |
|--------|--------|
| الخط غير مقروء وأنت غير متأكد | اتصل بالطبيب — لا تخمّن |
| الجرعة تتجاوز الحد الأقصى المعروف | اتصل للتأكيد قبل الصرف |
| المريض لديه حساسية من الدواء الموصوف | لا تصرف — أبلغ الطبيب فوراً |
| الدواء له تفاعل خطير مع دواء آخر للمريض | أبلغ الطبيب وانتظر تأكيده |
| الوصفة منتهية الصلاحية | لا تصرف |
| الوصفة تبدو مزوّرة أو معدّلة | لا تصرف وأبلغ المسؤول |
| طلب الكمية غير طبيعية من مادة خاضعة للرقابة | تحقق من الجهة المختصة |
